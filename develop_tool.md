## Ghostty(ターミナル)
```
brew install --cask ghostty
```

チートシートと再起動後のチュートリアル: [Ghostty](ghostty.md)

Ghosttyのconfigに下記を追加([doc](https://ghostty.org/docs/config))
```
config-file = ghostty.configへのパス
```

設定を変更した後はGhosttyを完全終了して再起動する。

基本操作:
- クイックターミナルの表示/非表示: `cmd+backquote`
- スプリット作成: `cmd+ctrl+h/j/k/l`
- スプリット移動: `cmd+shift+h/j/k/l`
- スプリットリサイズ: `cmd+alt+h/j/k/l`
- スプリットズーム: `cmd+shift+enter`
- スプリット均等化: `cmd+alt+=`
- 現在のスプリット/タブ/ウィンドウを閉じる: `cmd+ctrl+w`
- 閉じたスプリット/タブ/ウィンドウを戻す: `cmd+z`
- プロンプト単位の移動: `cmd+shift+[` / `cmd+shift+]`
