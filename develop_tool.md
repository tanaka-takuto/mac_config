# 開発ツール


## VSCode
```
brew install visual-studio-code
```

## warp(ターミナル)
```
brew install warp
```

## Ghostty(ターミナル)
```
brew install --cask ghostty
```

Ghosttyのconfigに下記を追加([doc](https://ghostty.org/docs/config))
```
config-file = ghostty.configへのパス
```
